Data Flow

서버가 HwpForge Markdown 자료를 읽어 OpenAI 답변, 근거, 비용을 표시합니다

1 HWP 원본들 인사규정, 보수 및 퇴직금 규정
2 HwpForge HWP → HWPX → Markdown 변환
3 Markdown 데이터 /data 폴더에 게시
4 브라우저 질문과 전송 범위만 보냄
5 API 프록시 선택된 Markdown을 서버에서 읽음
6 OpenAI 답변, 인용, 토큰 사용량 반환
7 화면 표시 답변, 근거, 토큰/비용 렌더링
서버의 Markdown 자료를 API에서 읽어 답변을 생성합니다.
