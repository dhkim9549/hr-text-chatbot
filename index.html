Data Flow

HwpForge로 만든 Markdown 자료를 OpenAI에 전달해 답변, 근거, 비용을 표시합니다

1 HWP 원본들 인사규정, 보수 및 퇴직금 규정
2 HwpForge HWP → HWPX → Markdown 변환
3 Markdown 데이터 /data 폴더에 게시
4 브라우저 Markdown 파일들을 불러옴
5 API 프록시 질문과 선택된 문서 전달
6 OpenAI 답변, 인용, 토큰 사용량 반환
7 화면 표시 답변, 근거, 토큰/비용 렌더링
이 폴더의 인사규정 Markdown 파일을 검색해 답변과 근거를 표시합니다.
